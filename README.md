# restful-api
RESTful API with NodeJS, TypeScript, Express, JWT.

## Project Setup

### Compiles for development

``` bash
yarn serve
```

### Lints files

``` bash
yarn lint
```

### Compiles for production

``` bash
yarn bulid
```
## Extral Command

### Generate apidoc

``` bash
apidoc -i src/routes -o public/apiDocs
```

### Git commit

``` bash
yarn commit
```
